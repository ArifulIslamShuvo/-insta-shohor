Difference between local storage and session storage

Local storage data is permanent but session storage data is temporary. Data must be deleted manually from local storage but in the case of session storage, if it is closed in browser or tab then all data in session storage is deleted. Local storage does not work for specific tabs, it works for all tabs, but session storage works for specific tabs, and data is lost when tabs are closed.

Both storage objects provide the same method and features:
  • setItem(key, value)-Store key / value pair.
  • getItem(key)- Get value by key.
  • removeItem(key) - Remove key by value.
  • clear() - Delete everything.
  • key(index) - Find the key to a specific position.
  • length The total number of items stored.
